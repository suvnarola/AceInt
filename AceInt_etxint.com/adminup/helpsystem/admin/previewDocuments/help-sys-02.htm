Creating your document

In the document browsing screen click on the "New Document" link. This will bring up the document editing screen displaying the new document that you have just created.

Setting your document properties

Set the documents properties in the blue box at the top of the page. Be sure to enter a short description and some keywords if you want your document to be able to be searched for. When you have entered all the documents information, click the "Update" button to the bottom right to save your changes.

Adding Document Contents

Content in the help system is added in topic sections. Each topic section can have several sub-topics. To add a topic section click the "Add sub section" button in the blue box under the document properties box. This will bring up a dalogue box that will let you input the Title and Content that you want. When you have entered the correct content into this box, you can press the "submit" button on the bottom right to add the content.

Adding subtopics

To add a sub-topic, click the "Add Sub-Section" button on the topic that you would like to add a sub-topic to. This will bring up a form like the one mentioned above. Fill it out in the same manner to add a sub-topic.

Editing document contents

To edit a topic or sub-topic click on the "Edit" button to the right of the topic that you wish to edit. In the form that comes up, edit any details that you wish, then click the submit button to save your changes.

To remove a topic, click the "delete" button to the right of the section that you wish to delete.

Previewing, Publishing and Removing

When you are developing a document you may wish to preview the document in the format that it will be published in. To do this click the "Preview" button. This will create a document identical to the one that will be created if you publish your document.

When you want to publish your document, you can do so using the "Publish" button. Note that this will create a live document from the data that you have entered. Any data that you change in the document will only change in the live document when you publish the document again.

If you have a published document that you wish to remove from the live document list, you can click the "Un-Publish" button to do so. This will remove the live version of the document while leaving the draft version intact.

If you delete a document you will remove both the live version and the draft version. Deletion cannot be undone.
